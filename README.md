> **Athletic Training + Heavy Lifting for Lean Muscle & 6-Pack Abs**

A comprehensive 4-month workout program specifically designed to build lean muscle volume while strengthening weak areas (back, shoulders, arms, and traps). This program combines athletic movements with heavy lifting to create an impressive, functional physique.

## 🌐 Live Website

**Visit the interactive workout guide:** [https://sivolko.github.io/sivolko-gymkhana/](https://sivolko.github.io/sivolko-gymkhana/)

## 📊 Program Overview

| **Aspect** | **Details** |
|------------|-------------|
| **Duration** | 4 Months |
| **Training Days** | 6 days/week (Sunday rest) |
| **Session Length** | 70-90 minutes |
| **Style** | Athletic + Heavy Weight Hybrid |
| **Target** | Lean muscle + 6-pack abs + Upper body strength |

## 🎯 Primary Goals

- ✅ **Build lean muscle volume** (2-4kg gain)
- ✅ **Develop visible 6-pack abs**
- ✅ **Strengthen weak areas** (back, shoulders, arms, traps)
- ✅ **Maintain athletic performance**
- ✅ **Achieve V-taper physique**
- ✅ **Total body fat loss** (7-10kg expected)

## 📅 Weekly Training Split

| **Day** | **Focus** | **Duration** | **Primary Exercises** |
|---------|-----------|--------------|----------------------|
| **Monday** | Back & Traps Power | 75 min | Deadlifts, Power Cleans, Weighted Pull-ups |
| **Tuesday** | Shoulders & Arms Athletic | 80 min | Push Press, Lateral Raises, Athletic Conditioning |
| **Wednesday** | Lower Body + Back Width | 70 min | Front Squats, RDL, Wide-Grip Pull-ups |
| **Thursday** | Upper Body Power + Traps | 85 min | Clean & Press, Heavy Rows, Explosive Circuits |
| **Friday** | Back & Arms Hypertrophy | 85 min | High Volume, Supersets, Pump Training |
| **Saturday** | Athletic + Weak Points | 90 min | Full Body Circuits, Specialization Work |
| **Sunday** | **Complete Rest** | - | Recovery, Mobility, Meal Prep |

## 🚀 Getting Started

1. Visit [https://sivolko.github.io/sivolko-gymkhana/](https://sivolko.github.io/sivolko-gymkhana/)

### **Option 2: Run Locally**
1. Clone the repository:
   ```bash
   git clone https://github.com/sivolko/sivolko-gymkhana.git
   ```

2. Navigate to the project directory:
   ```bash
   cd sivolko-gymkhana
   ```

3. Open `index.html` in your browser:
   ```bash
   # On macOS
   open index.html
   
   # On Windows
   start index.html
   
   # On Linux
   xdg-open index.html
   ```

### **Option 3: Print-Friendly Version**
1. Visit the website
2. Use browser's print function (Ctrl+P / Cmd+P)
3. Select "Save as PDF" for portable gym reference
4. Print specific workout tables as needed

## 🔄 Weekly Routine

### **Phase 1 (Weeks 1-4): Foundation**
- Learn all movement patterns
- Establish training rhythm
- Build baseline strength
- Adapt to volume

### **Phase 2 (Weeks 5-8): Intensity**
- Increase weights progressively
- Add complexity to movements
- Focus on weak point development
- Refine nutrition adherence

### **Phase 3 (Weeks 9-12): Specialization**
- Peak training intensity
- Maximum volume periods
- Advanced exercise variations
- Fine-tune diet for cutting

### **Phase 4 (Weeks 13-16): Refinement**
- Maintain strength gains
- Focus on definition
- Perfect form and technique
- Prepare for program completion

## 💡 Key Success Principles

### **Training**
- **Progressive Overload:** Increase weight/reps weekly
- **Form First:** Never compromise technique for weight
- **Consistency:** 6 days/week minimum adherence
- **Rest:** Adequate sleep (7-9 hours) for recovery

### **Nutrition**
- **Caloric Deficit:** 300-500 calories below maintenance
- **Protein Priority:** Hit protein targets daily
- **Meal Timing:** Eat around workouts for performance
- **Hydration:** 4-5 liters of water daily

### **Recovery**
- **Sleep:** 7-9 hours nightly
- **Stress Management:** Include relaxation techniques
- **Mobility:** Daily stretching and movement
- **Nutrition:** Anti-inflammatory foods
